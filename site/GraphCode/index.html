    This code will print a graph with networkx and matplotlib library and use the variables from the step before. Using the edgelist list from the code before to add the edges to the graph.

    import networkx as nx
import matplotlib.pyplot as plt

g = nx.Graph()
g = g.to_directed()

g.add_edges_from(edgelist)
pos = nx.spring_layout(g)

plt.figure(1)
nx.draw(g, pos)
nx.draw_networkx_labels(g, pos, label, font_size=8)

plt.axis("off")
plt.show()

    As the directed graph created, you will see your links nodes and edges.
